Hexo 博客技术栈

本站从零搭建到现在的完整技术栈，按层级整理如下。所有服务均为 零成本（仅域名年费）或 免费计划。

一、前端（博客框架）

技术 用途 版本
Hexo 静态博客生成器 7.x
AnZhiYu 主题 博客主题（已 fork 自定制） 1.6.14
Node.js Hexo 运行环境 22.x
Pug 主题模板引擎 —

核心插件

插件 用途
hexo-abbrlink 文章短链接（:category/:abbrlink.html）
hexo-generator-sitemap 生成 sitemap.xml（Google/Bing）
hexo-generator-baidu-sitemap 生成 baidusitemap.xml（百度）
hexo-generator-feed 生成 RSS/Atom 订阅
hexo-generator-robotstxt 生成 robots.txt
hexo-butterfly-envelope 信封样式留言板

二、托管与部署

服务 用途 费用
GitHub Pages 静态文件托管 免费
GitHub Actions CI/CD 自动构建部署 免费
Cloudflare CDN 加速、HTTPS、DNS 解析 免费（域名年费除外）
GitHub 源码仓库（SourceHexoBlog） 免费

三、评论系统

服务 用途 费用

Netlify Twikoo 后端云函数托管 免费
MongoDB Atlas 评论数据存储（M0 免费集群，512MB） 免费

四、统计分析

服务 用途 查看方式
不蒜子 busuanzi 站点 UV/PV、文章阅读量 直接在博客侧边栏和文章页显示
Google Analytics 完整的流量来源、用户行为分析 登录 Google Analytics 后台
Cloudflare Web Analytics 隐私友好的轻量级统计 登录 Cloudflare 仪表盘

五、SEO 与搜索控制台

平台 状态 用途
Google Search Console ✅ 已接入并验证 搜索排名监控、索引状态
Bing Webmaster Tools ✅ 已接入并验证 必应搜索排名
Yandex Webmaster ✅ 已接入并验证 俄罗斯搜索引擎
百度搜索资源平台 ⏸ 暂未接入 —

SEO 技术手段

  • URL 结构：:category/:abbrlink.html（category_map 映射英文 slug）
  • 结构化数据：JSON-LD（WebSite + BlogPosting / TechArticle / HowTo）
  • 内链体系：文章间互相引用，tags 标签聚合
  • Sitemap：sitemap.xml + baidusitemap.xml
  • 规范 URL：pretty_urls.trailing_index: false 去除 index.html 后缀
  • 文章 slug：核心文章使用含”how to”搜索意图的英文永久链接
