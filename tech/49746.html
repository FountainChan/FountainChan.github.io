Floci — 把 AWS 搬到你电脑里跑的本地 Docker 服务模拟器

硬核老王2026-07-142026-07-17

原文链接: https://www.toutiao.com/article/7660705575277216275/
原文标题: GitHub 又出王炸！AWS 居然能在自己电脑里免费搭！
原文作者: 硬核老王
抓取日期: 2026-07-14 12:00

如果你写过程序，或者听说过 AWS，那你一定知道：云服务很好用，但也是真的烧钱。

更麻烦的是，很多开发者只是想测试一下代码，却不得不开通云服务器、配置账号、申请密钥，一套流程下来，时间没了，钱也花了。

最近 GitHub 上冒出来一个新项目——Floci，就是冲着这个痛点来的。

它可以简单理解成：把 AWS 搬到你的电脑里。

不用注册账号，不用绑定信用卡，也不用担心误操作扣费，只需要运行一个 Docker，就能在本地模拟 AWS 的各种服务，像 S3、DynamoDB、Lambda 等都可以直接测试。开发完成后，再部署到真正的云服务器即可。

更关键的是，它并不是一个”玩具”。很多 AWS 官方工具、SDK、Terraform 等都能直接对接，开发流程几乎不用改，原来怎么写，现在还是怎么写。对于团队开发、自动化测试、CI/CD 来说，可以节省大量时间和云资源成本。

Floci 诞生的背景也很有意思。此前不少开发者都在使用类似工具，但后来很多高级功能开始收费，还需要登录账号、获取授权 Token。Floci 则直接打出一句口号：永久免费、完全开源、不需要账号、不限制功能。

对于普通人来说，你可能不会天天写代码，但如果以后想学习云计算、AWS、DevOps，或者准备进入程序员行业，这个项目就是一个免费的练手机会。不用担心误删云资源，也不用因为测试几次就收到一张账单。
